# Glossar — Beauty-Industrie-Begriffe

> Für Claude Code: damit Du Domain-Sprache verstehst, wenn sie in Code / UI / Admin-Texten auftaucht.

| Begriff                         | Bedeutung                                                                                                                           |
| ------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| **Backbar**                     | Produkte, die der Salon während einer Behandlung verbraucht (nicht verkauft), z. B. Farbe, Entwickler, Shampoo in der Waschstation. |
| **Retail**                      | Produkte, die an Kunden verkauft werden.                                                                                            |
| **Booth Rental / Booth Renter** | Modell, bei dem Stylists eigenständig sind und Platz (einen „Booth") im Salon mieten. Nicht Angestellte.                            |
| **Commission**                  | Provisionsmodell: Stylist bekommt % vom eigenen Service- oder Retail-Umsatz.                                                        |
| **Chair Rental**                | Synonym für Booth Rental.                                                                                                           |
| **Front Desk**                  | Empfangspersonal, typischerweise nicht bedienend.                                                                                   |
| **Rebook / Rebooking**          | Wiederholungstermin, meist zum gleichen Service und gleichen Stylisten.                                                             |
| **No-Show**                     | Kunde ist trotz Buchung nicht erschienen.                                                                                           |
| **Waitlist**                    | Liste von Kunden, die einspringen, falls ein Slot frei wird.                                                                        |
| **Walk-In**                     | Spontan-Kunde, der ohne Termin reinkommt (häufig Barber).                                                                           |
| **Gap-Filling**                 | Freie Zeiten zwischen Terminen durch Waitlist-Kunden o. ä. füllen.                                                                  |
| **Service-Formula**             | Mischverhältnis / Rezept, z. B. „30 g 8N + 10 g 9.01 + 40 g 6 %".                                                                   |
| **Patch Test**                  | Allergietest vor Farb- oder chemischer Behandlung. In UK/IE Pflicht.                                                                |
| **Consultation**                | Beratungstermin vor einer größeren Behandlung.                                                                                      |
| **SOAP Notes**                  | Medizinische Notizform: Subjective, Objective, Assessment, Plan.                                                                    |
| **Before/After**                | Vorher-/Nachher-Fotos zur Dokumentation (beliebt auf Social Media).                                                                 |
| **Tip Out / Tip Pool**          | System, bei dem Trinkgeld anteilig an Assistenten / Team abgegeben wird.                                                            |
| **Salon Suite**                 | Privater Mikro-Salon-Raum, oft in Suite-Gebäuden.                                                                                   |
| **Suite Life**                  | Geschäftsmodell für Solo-Beauty-Pros, die eigene Suite mieten.                                                                      |
| **Medspa / Med Spa**            | Medizinische Spa-Einrichtung, bietet z. B. Botox, Filler, Laser.                                                                    |
| **Injectable**                  | Injizierbare Behandlung (Botox, Filler). Usage-based Preis.                                                                         |
| **Usage-Based Pricing**         | Preis nach verbrauchter Menge, z. B. 1 ml Filler = 450 €.                                                                           |
| **Package / Course**            | Vorgekauftes Paket von z. B. 6 Behandlungen mit Rabatt.                                                                             |
| **Membership**                  | Monatliches Abonnement mit inkludierten Leistungen.                                                                                 |
| **Punch Card**                  | Physisch oder digital: nach 10. Termin der 11. gratis.                                                                              |
| **Loyalty Program**             | Sammelpunkte-/Stufen-System.                                                                                                        |
| **Deposit**                     | Anzahlung bei Buchung, um No-Shows zu reduzieren.                                                                                   |
| **Cancellation Fee**            | Gebühr bei zu später Stornierung.                                                                                                   |
| **Round-Robin Booking**         | Verteilt Buchungen automatisch auf mehrere Stylists.                                                                                |
| **Resource Booking**            | Buchung eines Raums / Geräts (statt Staff), z. B. Laser-Gerät.                                                                      |
| **Intake Form**                 | Anamnese-Formular vor Behandlung.                                                                                                   |
| **Consent Form**                | Einwilligungserklärung (z. B. für Botox, bei Minderjährigen auch Sorgerecht).                                                       |
| **Color Bar**                   | Bereich im Salon, wo Farben angemischt werden.                                                                                      |
| **Wash Station / Wash Basin**   | Waschplatz.                                                                                                                         |
| **Blow-Out**                    | Haarstyling mit Föhn, häufig standalone buchbar.                                                                                    |
| **Balayage / Highlights**       | Haarfärbetechniken.                                                                                                                 |
| **Keratin / Brazilian**         | Glättungs-Behandlung.                                                                                                               |
| **Lash Lift / Lash Ext.**       | Wimpernbehandlungen.                                                                                                                |
| **Microblading**                | Augenbrauen-Permanent-Make-up-Technik.                                                                                              |
| **Pedicure / Manicure**         | Standard-Nagelbehandlungen.                                                                                                         |
| **Gel-X / Acrylic / Dip**       | Nagelverlängerungs-Techniken.                                                                                                       |
| **Pressed-on Nails**            | Künstliche Nägel zum Aufkleben.                                                                                                     |
| **Brow Bar**                    | Spezialisiertes Studio nur für Augenbrauen.                                                                                         |
| **Barber Shop**                 | Herrenfriseur, oft Walk-in-fokussiert.                                                                                              |
| **Fade**                        | Haarschnitt-Technik Barber.                                                                                                         |
| **Beard Sculpt**                | Bart-Modelling.                                                                                                                     |
| **Tattoo Flash**                | Vorgezeichnete Tattoos, schnell buchbar.                                                                                            |
| **Flash Day**                   | Tattoo-Event mit Sonderpreisen.                                                                                                     |
| **Piercing**                    | Durchstechen, oft walk-in.                                                                                                          |
| **Spa Circuit**                 | Wellness-Pass mit Sauna, Pool etc.                                                                                                  |
| **Locker / Cabin**              | Umkleide/Ruhezone in Spa.                                                                                                           |
| **Class**                       | Mehrpersonen-Kurs (Yoga, Pilates — Mindbody-Domäne).                                                                                |
| **Package**                     | Kombination mehrerer Services zum Paketpreis.                                                                                       |
| **Upsell / Cross-Sell**         | Verkauf zusätzlicher Services / Produkte bei Buchung oder Checkout.                                                                 |
| **Client Lifetime Value (CLV)** | Gesamtumsatz eines Kunden über die Beziehungsdauer.                                                                                 |
| **Churn**                       | Verlorene Kunden (Salon-Seite: nicht mehr gebucht).                                                                                 |
| **Chair Time**                  | Zeit, die eine Stylist:in mit bezahlter Arbeit am Stuhl verbringt.                                                                  |
| **Occupancy / Utilization**     | Auslastung, belegte vs. verfügbare Slot-Stunden.                                                                                    |
| **TSE**                         | Technische Sicherheitseinrichtung — Deutschland, Kassensicherung.                                                                   |
| **RKSV**                        | Registrierkassensicherheitsverordnung — Österreich.                                                                                 |
| **DSFinV-K**                    | Digitale Schnittstelle Finanzverwaltung für Kassensysteme (DE).                                                                     |
| **PSD2**                        | EU-Zahlungsdienstrichtlinie, SCA-Pflicht.                                                                                           |
| **SCA**                         | Strong Customer Authentication.                                                                                                     |
| **PCI-DSS**                     | Kartenzahlungs-Sicherheitsstandard.                                                                                                 |
| **HIPAA**                       | US-Gesundheitsdatenschutz-Gesetz.                                                                                                   |
| **BAA**                         | Business Associate Agreement — HIPAA-Pflichtvertrag.                                                                                |
| **PHI**                         | Protected Health Information (HIPAA).                                                                                               |
| **DSGVO / GDPR**                | EU-Datenschutz-Grundverordnung.                                                                                                     |
| **DPA / AVV**                   | Data Processing Agreement / Auftragsverarbeitungsvertrag.                                                                           |
| **KYB**                         | Know-Your-Business (Onboarding Salon = Merchant).                                                                                   |
| **MCC 7230**                    | Merchant-Category-Code für „Beauty / Barber Shops".                                                                                 |
| **MCC 7298**                    | „Health and Beauty Spas".                                                                                                           |
| **ZUGFeRD / XRechnung**         | E-Rechnung-Formate DE.                                                                                                              |
| **PEPPOL**                      | EU E-Invoice-Netzwerk.                                                                                                              |
| **DATEV**                       | DE-Buchhaltungs-Standard (Dateiformat + Tool-Ökosystem).                                                                            |
| **GoBD**                        | Grundsätze zur ordnungsmäßigen Buchführung (DE).                                                                                    |
